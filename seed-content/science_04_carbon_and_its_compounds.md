# Carbon and its Compounds - Learning Path

## 1. Concept Introduction

### What Makes Carbon Special?

**Carbon** is a unique element that forms the basis of all living organisms. It is the fourth most abundant element in the universe and forms more compounds than any other element.

**Everyday Examples:**
- 🍞 **Food:** Carbohydrates, proteins, fats (all contain carbon)
- 👕 **Clothing:** Cotton, wool, synthetic fibres (carbon-based)
- ⛽ **Fuel:** Petrol, diesel, CNG, LPG (hydrocarbons)
- 💊 **Medicines:** Most drugs are carbon compounds
- 📱 **Plastics:** Buckets, bags, toys (polymers)

### The Carbon Atom:

```
CARBON ATOM STRUCTURE

    Atomic Number: 6
    Electronic Configuration: 2, 4
    Valency: 4 (Tetravalent)
    
         ●●    ← K shell (2 electrons)
        ●  ●
       ●  C  ●  ← L shell (4 valence electrons)
        ●  ●
         ●●
    
    Needs 4 more electrons to complete octet
    Forms 4 covalent bonds
```

**Key Components:**

| Property | Value | Significance |
|----------|-------|--------------|
| **Atomic Number** | 6 | 6 protons, 6 electrons |
| **Electronic Configuration** | 2, 4 | 4 valence electrons |
| **Valency** | 4 | Forms 4 bonds (tetravalent) |
| **Bond Type** | Covalent | Shares electrons |
| **Catenation** | Yes | Forms long chains |
| **Allotropy** | Yes | Diamond, Graphite, Fullerene |

## 3. Concept Explanations

### 4.1 Bonding in Carbon - The Covalent Bond

Carbon cannot form ionic bonds easily because:
- **Losing 4 electrons** requires too much energy (C⁴⁺ unstable)
- **Gaining 4 electrons** is difficult (C⁴⁻ unstable, nucleus can't hold 10 electrons)

**Solution:** Carbon **shares** electrons with other atoms → **Covalent Bond**

#### Types of Covalent Bonds:

**1. Single Bond:** Sharing of one electron pair

```
HYDROGEN MOLECULE (H₂)

H·  +  ·H  →  H:H  or  H-H

Each H shares 1 electron
Both achieve helium configuration (2 electrons)
```

**2. Double Bond:** Sharing of two electron pairs

```
OXYGEN MOLECULE (O₂)

:Ö·  +  ·Ö:  →  :O::O:  or  O=O

Each O shares 2 electrons
Both achieve neon configuration (8 electrons)
```

**3. Triple Bond:** Sharing of three electron pairs

```
NITROGEN MOLECULE (N₂)

·N·  +  ·N·  →  :N:::N:  or  N≡N
 ··       ··

Each N shares 3 electrons
Both achieve neon configuration (8 electrons)
```

---

#### Electron Dot Structures of Carbon Compounds:

**Methane (CH₄):**

```
    H
    |
H - C - H    or    H:C:H
    |                  |
    H                  H

Carbon shares 4 electrons with 4 hydrogen atoms
Each bond is a single covalent bond
Carbon achieves neon configuration (8 electrons)
Each H achieves helium configuration (2 electrons)
```

**Water (H₂O):**

```
    H
    |
H - O:    or    H:Ö:H
    ¨

Oxygen shares 2 electrons (one with each H)
Oxygen has 2 lone pairs
```

**Ammonia (NH₃):**

```
    H
    |
H - N:    or    H:N̈:H
    |            |
    H            H

Nitrogen shares 3 electrons
Nitrogen has 1 lone pair
```

---

#### Properties of Covalent Compounds:

| Property | Explanation | Example |
|----------|-------------|---------|
| **Low melting/boiling points** | Weak intermolecular forces | H₂O boils at 100°C |
| **Poor conductors** | No free electrons/ions | Sugar solution doesn't conduct |
| **Soft solids** | Weak forces between molecules | Wax, sulphur |
| **Solubility** | Soluble in organic solvents, insoluble in water | I₂ in CCl₄ |

**Exceptions:**
- **Diamond:** Very high melting point (giant structure)
- **Graphite:** Conducts electricity (free electrons)

### 4.2 Allotropes of Carbon

Carbon exists in different forms called **allotropes** - same element, different structures and properties.

```
ALLOTROPES OF CARBON

    CRYSTALLINE          AMORPHOUS
    ┌─────────┬─────────┬─────────┐
    │         │         │         │
  Diamond   Graphite  Fullerene  Coal, Charcoal
    │         │         │         (Impure forms)
    │         │         │
  Hard     Soft     Soccer ball
  Insulator Conductor   shape
```

#### 1. Diamond:

```
DIAMOND STRUCTURE

       C
      /|\
     / | \
    C--C--C
    |\ | /|
    | \|/ |
    C--C--C
     \ | /
      \|/
       C

Each carbon bonded to 4 others (tetrahedral)
3D network structure
Very hard, transparent, brilliant
```

**Properties:**
- Hardest natural substance
- Transparent, brilliant lustre
- Poor conductor of electricity
- High melting point (3550°C)
- Density: 3.5 g/cm³

**Uses:**
- Jewellery
- Cutting and drilling tools
- Surgical instruments

#### 2. Graphite:

```
GRAPHITE STRUCTURE

    Layer 1:    C - C - C - C
                |   |   |   |
                C - C - C - C
    
    Layer 2:    C - C - C - C
                |   |   |   |
                C - C - C - C
    
    Weak forces between layers
    Layers can slide over each other
```

**Properties:**
- Soft, slippery (layers slide)
- Grey-black, metallic lustre
- Good conductor of electricity
- High melting point (3730°C)
- Density: 2.2 g/cm³

**Uses:**
- Pencil leads
- Lubricant
- Electrodes in batteries
- Crucibles for melting metals

### 4.3 Versatile Nature of Carbon

Carbon forms millions of compounds due to:

#### 1. Catenation:

**Self-linking property** - Carbon atoms can link to form:

```
TYPES OF CARBON CHAINS

STRaight Chain:    C - C - C - C - C - C
                   (Hexane)

Branched Chain:        C
                       |
                   C - C - C - C
                       |
                       C
                   (Isopentane)

Closed Chain:      C - C
                   /     \
                  C       C
                   \     /
                    C - C
                  (Cyclohexane)
```

**Other elements showing catenation:**
- Sulphur (S₈ rings)
- Phosphorus (P₄)
- Silicon (limited)

---

#### 2. Tetravalency:

Carbon has **4 valence electrons** → can form **4 bonds**

```
CARBON'S TETRAVALENCY

    H
    |
H - C - H    (Methane - 4 single bonds)
    |
    H

    H   H
    |   |
H - C - C - H    (Ethane - each C has 4 bonds)
    |   |
    H   H
```

---

#### 3. Multiple Bonding:

Carbon can form **single, double, or triple bonds**

```
MULTIPLE BONDS IN CARBON

Single:    H - C - C - H    (Ethane, C₂H₆)
           |   |
           H   H

Double:    H   H
            \ /
             C = C    (Ethene, C₂H₄)
            / \
           H   H

Triple:    H - C ≡ C - H    (Ethyne, C₂H₂)
```

### 4.4 Organic Compounds

#### Hydrocarbons:

Compounds containing **only carbon and hydrogen**.

**Classification:**

```
HYDROCARBONS

┌──────────────────┬──────────────────┐
│   SATURATED      │   UNSATURATED    │
│   (Alkanes)      │                  │
│                  │  ┌────────┬──────┤
│  C - C single    │  │ Alkenes│ Alkynes│
│  bonds only      │  │ C=C    │ C≡C   │
│  General: CₙH₂ₙ₊₂│  │ CₙH₂ₙ  │ CₙH₂ₙ₋₂│
└──────────────────┴─────────┴──────┘
```

**Examples:**

| Type | Name | Formula | Structure |
|------|------|---------|-----------|
| **Alkane** | Methane | CH₄ | CH₄ |
| | Ethane | C₂H₆ | CH₃-CH₃ |
| | Propane | C₃H₈ | CH₃-CH₂-CH₃ |
| **Alkene** | Ethene | C₂H₄ | CH₂=CH₂ |
| | Propene | C₃H₆ | CH₃-CH=CH₂ |
| **Alkyne** | Ethyne | C₂H₂ | HC≡CH |
| | Propyne | C₃H₄ | CH₃-C≡CH |

---

#### Functional Groups:

**Specific atoms/groups** that determine properties of organic compounds.

| Functional Group | Structure | Class | Example |
|-----------------|-----------|-------|---------|
| **Halo** | -X (F, Cl, Br, I) | Haloalkane | CH₃Cl (Chloromethane) |
| **Alcohol** | -OH | Alcohol | CH₃OH (Methanol) |
| **Aldehyde** | -CHO | Aldehyde | HCHO (Methanal) |
| **Ketone** | >C=O | Ketone | CH₃COCH₃ (Propanone) |
| **Carboxylic Acid** | -COOH | Acid | CH₃COOH (Ethanoic acid) |

---

#### Homologous Series:

A series of compounds with:
- **Same functional group**
- **Similar chemical properties**
- **Gradation in physical properties**
- **Successive members differ by -CH₂-**

**Example - Alkanes:**

```
ALKANE HOMOLOGOUS SERIES

CH₄    →    C₂H₆    →    C₃H₈    →    C₄H₁₀
Methane     Ethane       Propane      Butane
            (-CH₂-)      (-CH₂-)      (-CH₂-)

Mass difference: 14u between each member
```

### 4.5 Chemical Properties of Carbon Compounds

#### 1. Combustion:

Burning in oxygen to produce CO₂, H₂O, heat and light.

**General Equation:**
$$ \text{Hydrocarbon} + \text{O}_2 \rightarrow \text{CO}_2 + \text{H}_2\text{O} + \text{Heat} + \text{Light} $$

**Examples:**

**Methane:**
$$ \text{CH}_4 + 2\text{O}_2 \rightarrow \text{CO}_2 + 2\text{H}_2\text{O} + \text{Heat} + \text{Light} $$

**Ethane:**
$$ 2\text{C}_2\text{H}_6 + 7\text{O}_2 \rightarrow 4\text{CO}_2 + 6\text{H}_2\text{O} + \text{Heat} + \text{Light} $$

**Observations:**
- Blue flame (complete combustion)
- Yellow sooty flame (incomplete combustion, limited oxygen)

---

#### 2. Oxidation:

Conversion of alcohols to carboxylic acids.

**Ethanol to Ethanoic Acid:**

```
    [O]                    [O]
CH₃CH₂OH  →  CH₃CHO  →  CH₃COOH
Ethanol      Ethanal     Ethanoic Acid
(Alcohol)    (Aldehyde)  (Acid)

Oxidizing agents:
- Alkaline KMnO₄
- Acidified K₂Cr₂O₇
```

**Equation:**
$$ \text{CH}_3\text{CH}_2\text{OH} + 2[\text{O}] \xrightarrow{\text{KMnO}_4} \text{CH}_3\text{COOH} + \text{H}_2\text{O} $$

---

#### 3. Addition Reactions:

Unsaturated compounds add atoms across double/triple bonds.

**Hydrogenation (Hardening of Oils):**

```
    Ni catalyst
C₂H₄ + H₂  ──────→  C₂H₆
Ethane          Ethane
(Unsaturated)   (Saturated)

150°C

Vegetable Oil + H₂ → Vanaspati Ghee
(Unsaturated fat)   (Saturated fat)
```

**Uses:**
- Making margarine from vegetable oils
- Petrochemical industry

---

#### 4. Substitution Reactions:

One atom/group replaces another.

**Chlorination of Methane (in sunlight):**

```
    Sunlight
CH₄ + Cl₂  ──────→  CH₃Cl + HCl
Methane            Chloromethane

Further substitution:
CH₃Cl + Cl₂ → CH₂Cl₂ (Dichloromethane)
CH₂Cl₂ + Cl₂ → CHCl₃ (Trichloromethane/Chloroform)
CHCl₃ + Cl₂ → CCl₄ (Tetrachloromethane)
```

**Note:** Saturated compounds show substitution; unsaturated show addition.

### 4.6 Important Carbon Compounds

#### 1. Ethanol (C₂H₅OH):

**Properties:**
- Colourless liquid
- Pleasant odour
- Boiling point: 78°C
- Soluble in water
- Neutral to litmus

**Chemical Properties:**

**(a) Reaction with Sodium:**
$$ 2\text{Na} + 2\text{C}_2\text{H}_5\text{OH} \rightarrow 2\text{C}_2\text{H}_5\text{ONa} + \text{H}_2 $$
(Sodium ethoxide + Hydrogen)

**(b) Dehydration (with conc. H₂SO₄ at 170°C):**
$$ \text{C}_2\text{H}_5\text{OH} \xrightarrow{\text{conc. H}_2\text{SO}_4, 170°C} \text{C}_2\text{H}_4 + \text{H}_2\text{O} $$
(Ethanol → Ethene)

**(c) Esterification:**
$$ \text{C}_2\text{H}_5\text{OH} + \text{CH}_3\text{COOH} \xrightarrow{\text{H}^+} \text{CH}_3\text{COOC}_2\text{H}_5 + \text{H}_2\text{O} $$
(Ethanol + Ethanoic acid → Ethyl ethanoate ester)

**Uses:**
- Alcoholic beverages
- Antiseptic (tincture iodine)
- Fuel (mixed with petrol)
- Solvent (medicines, perfumes)
- Antifreeze

---

#### 2. Ethanoic Acid (CH₃COOH):

**Properties:**
- Colourless liquid
- Pungent smell
- Sour taste
- Boiling point: 118°C
- Soluble in water
- Weak acid (turns blue litmus red)

**5-8% solution in water = Vinegar**

**Chemical Properties:**

**(a) Esterification:**
$$ \text{CH}_3\text{COOH} + \text{C}_2\text{H}_5\text{OH} \xrightarrow{\text{H}^+} \text{CH}_3\text{COOC}_2\text{H}_5 + \text{H}_2\text{O} $$
(Ethanoic acid + Ethanol → Ethyl ethanoate + Water)

**Esters:** Sweet-smelling compounds used in perfumes, flavouring agents

**(b) Reaction with Base (Neutralization):**
$$ \text{CH}_3\text{COOH} + \text{NaOH} \rightarrow \text{CH}_3\text{COONa} + \text{H}_2\text{O} $$
(Sodium ethanoate)

**(c) Reaction with Carbonates:**
$$ 2\text{CH}_3\text{COOH} + \text{Na}_2\text{CO}_3 \rightarrow 2\text{CH}_3\text{COONa} + \text{H}_2\text{O} + \text{CO}_2 $$

**(d) Reaction with Hydrogen Carbonates:**
$$ \text{CH}_3\text{COOH} + \text{NaHCO}_3 \rightarrow \text{CH}_3\text{COONa} + \text{H}_2\text{O} + \text{CO}_2 $$

**Test for Carboxylic Acid:** Evolves CO₂ with NaHCO₃ (turns lime water milky)

**Uses:**
- Vinegar (food preservative)
- Manufacture of plastics, dyes
- Rubber industry
- Perfumes

---

#### 3. Soaps and Detergents:

**Soap:** Sodium/potassium salt of long-chain carboxylic acids.

**Structure:**

```
SOAP MOLECULE

    Hydrophilic Head    Hydrophobic Tail
    (Ionic - COO⁻Na⁺)   (Long hydrocarbon chain)
          ●━━━━━━━━━━━━━━━━━━━━
          │
    Water-loving         Oil-loving
    (Dissolves in water) (Dissolves in oil)
```

**Cleansing Action:**

```
CLEANSING ACTION OF SOAP

Step 1: Soap molecules surround oil droplet

        Tail (in oil)
        ↓
    ●━━━●━━━●
   /         \
  ●  Oil     ●  ← Hydrophilic heads (in water)
   \  drop   /
    ●━━━●━━━●
        ↑
        Tail

Step 2: Forms MICELLE (spherical structure)

Step 3: Micelles suspended in water
        Oil droplet washed away
```

**Micelle Formation:**
- Hydrophobic tails trap oil/grease
- Hydrophilic heads face outward (in water)
- Forms stable emulsion
- Oil droplets washed away

---

**Soap vs Detergent:**

| Soap | Detergent |
|------|-----------|
| Sodium salt of fatty acid | Sodium salt of sulphonic acid |
| Doesn't work in hard water | Works in hard water |
| Biodegradable | Some non-biodegradable |
| Mild cleansing | Strong cleansing |

**Problem with Hard Water:**
$$ 2\text{C}_{17}\text{H}_{35}\text{COONa} + \text{Ca}^{2+} \rightarrow (\text{C}_{17}\text{H}_{35}\text{COO})_2\text{Ca} \downarrow + 2\text{Na}^+ $$
(Insoluble scum formed)

**Detergents work in hard water** - calcium salts are soluble.

### Example 4: Identifying Reaction Types

**Question:** Identify the type of reaction:

### Example 5: Soap and Hard Water

**Question:** Explain why soap doesn't work well in hard water. Write the chemical equation.

**Step-by-Step Solution:**

**Explanation:**

| Step | Explanation |
|------|-------------|
| 1 | Hard water contains Ca²⁺ and Mg²⁺ ions |
| 2 | Soap is sodium salt of fatty acid (RCOONa) |
| 3 | Ca²⁺/Mg²⁺ displace Na⁺ from soap |
| 4 | Insoluble calcium/magnesium soap formed |
| 5 | This precipitate is called **scum** |
| 6 | Soap wasted in forming scum |
| 7 | Less soap available for cleansing |

**Equation:**
$$ 2\text{C}_{17}\text{H}_{35}\text{COONa} + \text{Ca}^{2+} \rightarrow (\text{C}_{17}\text{H}_{35}\text{COO})_2\text{Ca} \downarrow + 2\text{Na}^+ $$
(Sodium stearate) (Calcium stearate - scum)

**Solution:** Use detergents instead (calcium salts are soluble)

**Q57.** (a) What is a homologous series? Explain with example of alkanes.

<details>
<summary>✓ Show Answer</summary>

- Same general formula
- Similar chemical properties
- Gradation in physical properties

```
CH₄ → C₂H₆ → C₃H₈ → C₄H₁₀ → C₅H₁₂
Methane  Ethane  Propane  Butane  Pentane
```

-CH₂- unit = 12 + 2 = **14u**

**Q61.** (a) What are the chemical properties of carbon compounds? Explain any three.
(b) Why do saturated compounds undergo substitution while unsaturated undergo addition?

<details>
<summary>✓ Show Answer</summary>

**Answer:**

**(a) Chemical Properties:**

**1. Combustion:**
- Burning in oxygen
- Produces CO₂, H₂O, heat, light
$$ \text{CH}_4 + 2\text{O}_2 \rightarrow \text{CO}_2 + 2\text{H}_2\text{O} + \text{Heat} $$

**2. Oxidation:**
- Addition of oxygen or removal of hydrogen
- Alcohols → Acids
$$ \text{C}_2\text{H}_5\text{OH} \xrightarrow{[\text{O}]} \text{CH}_3\text{COOH} $$

**3. Addition Reactions:**
- Unsaturated compounds add atoms
- H₂ addition (hydrogenation)
$$ \text{C}_2\text{H}_4 + \text{H}_2 \rightarrow \text{C}_2\text{H}_6 $$

**4. Substitution Reactions:**
- Atom/group replaced by another
- Chlorination of methane
$$ \text{CH}_4 + \text{Cl}_2 \rightarrow \text{CH}_3\text{Cl} + \text{HCl} $$

**(b) Reason for Different Reactions:**

**Saturated (Substitution):**
- All bonds are single
- No scope for addition
- Must replace existing atom
- Requires energy (UV light)

**Unsaturated (Addition):**
- Has double/triple bonds
- π-bonds are weak
- Can break and add atoms
- More reactive
</details>
